<img> ‚Äî Image

Purpose: embed images; void element (no closing tag). Use alt for accessibility.

Live demo
Common attributes: src, alt, width, height, loading

Example

<img src="bag.png" alt="Blue bag" width="200" loading="lazy">
EN: Inline element to show an image. Always provide meaningful alt text for accessibility.

<a> ‚Äî Anchor / Link

Purpose: create hyperlinks. Can navigate to other resources or trigger downloads.

Open MDN (new tab)
Try target=_blank + rel="noopener"

Example

<a href="https://example.com" target="_blank" rel="noopener" title="Example site">Visit example</a>
Key attributes: href, target, rel, download, title
EN: Creates a hyperlink. Use rel="noopener" with target="_blank" for security.

<ul>, <ol>, <li> ‚Äî Lists

Purpose: group related items. <ul> = unordered, <ol> = ordered, each item in <li>.

  • HTML
  • CSS
  • JavaScript
  1. Step A
  2. Step B
  3. Step C
Use CSS list-style-type to change markers.

Example

<ul style="list-style-type:square;">
  <li>HTML</li>
  <li>CSS</li>
</ul>

<ol type="a">
  <li>First</li>
  <li>Second</li>
</ol>
EN: Use lists for groups of related items; keep semantics correct for accessibility.

Quick Tips üîß

  1. Always use semantic tags for accessibility and SEO.
  2. Provide alt for images and descriptive title when helpful.
  3. For links opening in new tabs, add rel="noopener noreferrer".
  4. Style lists via CSS; avoid using lists purely for layout when not semantically appropriate.
